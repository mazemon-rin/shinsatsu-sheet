# 診察メモメーカー

医師に見せる体調・既往歴まとめシートを、Webブラウザだけで作成できるアプリです。

このサービスは診断を行うものではありません。診察時に症状や経過を説明しやすくするためのメモ作成ツールです。

## 公開ページ

https://mazemon-rin.github.io/shinsatsu-sheet/

## 概要

病院やクリニックで診察を受ける際、

* いつから症状があるのか
* どのような経過をたどったのか
* どんな薬を飲んでいるのか
* どんな病気にかかったことがあるのか
* 生活や仕事にどのような影響があるのか

を短時間で正確に伝えることは意外と大変です。

診察メモメーカーは、受診前に情報を整理し、医師へ伝えやすくすることを目的として開発しました。

## 主な機能

### 基本情報整理

* 氏名
* 年齢
* 作成日
* 受診科
* 主治医に伝えたい目的

### 症状整理

* 主症状の選択
* 受診科に合わせた症状候補
* 症状の強さ
* 頻度
* 悪化する場面
* 楽になる場面
* 症状経過の時系列記録

### 既往歴・治療中の病気

* 既往歴
* 現在治療中の病気
* 治療内容
* 現在の状態

### 服薬・検査結果

* 服薬情報
* 薬の種類
* 飲み始めた時期
* 気になる副作用
* 検査・診察結果

### 診察サポート

* 医師へ相談したい内容の整理
* A4印刷用プレビュー
* AI要約用テキスト作成
* コピー機能

## データの扱い

入力内容は、利用者が使っているPC・タブレット・スマホなどのブラウザ内で扱います。

このアプリ自体は、入力内容をサーバーへ送信しません。

ただし、AI要約用テキストをChatGPTなどの外部サービスへ貼り付ける場合は、個人情報や医療情報を外部サービスに入力することになります。氏名や住所など、不要な個人情報は削除してから利用してください。

## 利用方法

1. 必要項目を入力します。
2. 画面右側または下部のプレビューを確認します。
3. 必要に応じて印刷します。
4. AI要約を使う場合は、AI要約用テキストを作成し、自分でChatGPTなどに貼り付けます。

## 対象者

* 症状や経過をうまく説明したい方
* 慢性疾患や既往歴を整理したい方
* 高血圧、糖尿病、耳鳴り、めまい、不眠、不安などを相談したい方
* 心療内科・精神科・内科などの受診前に情報をまとめたい方
* ご家族の受診をサポートする方
* 医療機関へ説明するのが苦手な方

## 注意事項

本アプリは診察時の説明補助および情報整理を目的としたツールです。

診断、治療、医療判断、薬の変更指示を行うものではありません。

表示内容やAI要約用テキストは、医師に伝えるためのメモとして利用してください。

強い胸痛、息苦しさ、麻痺、ろれつが回らない、激しい頭痛、高度な血圧上昇など緊急性の高い症状がある場合は、すぐに医療機関へ相談してください。
